<h1 align="center" style="font-weight: normal;"><b>Hyve</b></h1>
<p align="center">Golang API for creating headless bots on Hytale</p>

Hyve is a Golang library for building headless bots and automated clients for the game Hytale. Its primary goal is to accurately replicate the network communication between the official Hytale client and servers, enabling developers to programatically connect, interact with worlds, test gameplay systems, and simulate players without running the full game client.
